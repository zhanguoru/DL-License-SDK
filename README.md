# Quick Start

## Install Dependencies

```bash
pip install PySide6
pip install cryptography
pip install requests
```

## Configure Application

Copy `License_Core/app.json` generated by the licensing platform into:

```text
License_Core/app.json
```

## Enable License Protection

```python
from License_Core.protect import protect

APP_API_KEY = "APK_xxxxxxxxxxxxxxxxx"

protect(APP_API_KEY)
```

## Get Device ID (Optional)

```python
from License_Core.device import get_device_fingerprint

device_id = get_device_fingerprint()
```

## Force Reactivation (Optional)

```python
protect(APP_API_KEY, force_activate=True)
```

---

### Features

* License key activation
* Device binding
* Offline license verification
* Expiration checking
* Anti-tamper verification
* Runtime protection
* Usage reporting

Only one line of code is required:

```python
protect(APP_API_KEY)
```
